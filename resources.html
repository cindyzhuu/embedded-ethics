Ethics Assignments

Four comprehensive ethics activities designed to integrate seamlessly into your AP CS A curriculum. Each assignment includes student handouts, teacher guides, and assessment rubrics.

Assignment 1: Objects and Methods

Students create a social media recommendation system while exploring ethical implications of content algorithms and user data analysis.

CS Topics Covered:

  • Objects and methods
  • User data structures
  • Recommendation algorithms

Ethics Focus:

  • Social media algorithms
  • User privacy and data collection
  • Content recommendation ethics

Assignment 2: Selection and Iteration

Students explore conditional statements while examining how algorithms can create bias in hiring algorithm analysis.

CS Topics Covered:

  • If-else statements
  • Boolean expressions
  • Conditional logic

Ethics Focus:

  • Algorithmic bias
  • Fairness in automated decisions
  • Discrimination in code

Assignment 3: Class Creations

Students design a person and select attributes for use in a medical app.

CS Topics Covered:

  • Classes, objects
  • Object interaction
  • Encapsulation

Ethics Focus:

  • Data privacy
  • Potential for bias

Assignment 4: Data Structures

Students design a framework to collect data, evaluate, and recommend school snacks.

CS Topics Covered:

  • Arrays and lists
  • Data processing
  • Array traversal

Ethics Focus:

  • Value tradeoffs
  • Student prioritization
